# 변경 이력

전체 변경 이력(Keep a Changelog 형식)은 리포지토리에 있습니다:

- **[GitHub의 CHANGELOG.md](https://github.com/devslab-kr/devslab-kit/blob/main/CHANGELOG.md)**
  — 모든 변경 사항과 마이그레이션 노트.
- **[GitHub Releases](https://github.com/devslab-kr/devslab-kit/releases)** —
  발행된 릴리스와 노트.

라이브러리 메이저는 Spring Boot 메이저를 따릅니다: `4.x.y`는 Spring Boot 4.x 대상.
